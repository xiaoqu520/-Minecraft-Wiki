# 🎮 主菜单指南

{% hint style="success" %}
**TrMenu**：手持 **时钟** 或输入 `/cd`、`/cd2` 打开两页主菜单，覆盖大部分日常功能。
{% endhint %}

## 👤 适合谁

不想记大量命令、希望通过图形界面传送、购物、管理领地的玩家。

| 命令 | 菜单 |
|------|------|
| `/cd` | 主菜单（第 1 页） |
| `/cd2` | 主菜单（第 2 页） |

## 主菜单第 1 页（`/cd`）

| 入口 | 功能 | 文档 |
|------|------|------|
| 功能菜单 | 每日签到、图鉴、称号 | [玩法百科](../gameplay/README.md) |
| 商城购物 | 全球市场、称号、宠物、时装 | [商城购物](#商城购物) |
| 领地系统 | 领地列表与管理 | [领地系统](../gameplay/residence.md) |
| 会员界面 | VIP/SVIP/MVP、点券充值 | [会员权益](../membership/README.md) |
| 玩法菜单 | 含 **全球市场**（`market gui`）等 | [玩法总览](../gameplay/README.md) |
| 世界旅行 | 回城、家园、各世界传送 | [世界旅行](../worlds/teleport.md) |
| 世界地标 | 玩家地标 | [玩家地标](../gameplay/landmarks.md) |
| 公会 | 创建/加入公会 | [公会系统](../gameplay/guild.md) |
| 死亡记录 | 墓碑列表 | [死亡墓碑](../gameplay/graves.md) |
| 好友系统 | 好友列表 | [好友系统](../gameplay/friends.md) |
| 下一页 | 跳转 `/cd2` | — |
| 指令帮助 | 新手教程 NPC | `plk openNpc 3` |
| 关闭菜单 | 关闭界面 | — |

## 商城购物

从主菜单进入 **商城购物** 子菜单：

| 入口 | 命令 | 说明 |
|------|------|------|
| 系统商店 | `/shopgui` | ShopGUIPlus（**非**玩家 QuickShop 市场） |
| 称号商城 | `/plt shop` | 称号购买 |
| 宠物召唤 | `/cpp shop` | CompanionsPlus 商店 |

详见 [系统商店](../gameplay/shops.md)、[称号系统](../gameplay/titles.md)、[宠物系统](../gameplay/pets.md)。

## 会员与充值

主菜单 **会员界面** → [新会员菜单](../membership/overview.md)：

- VIP / SVIP / MVP 月卡、季卡  
- **点券充值**（微信 / 支付宝）  

充值前请阅读 [赞助与消费条例](../legal/赞助条例.md)。
