# 🔐 账号与安全

{% hint style="info" %}
安途使用 **AuthMe** 独立游戏密码，与 Mojang 正版账号 **不是同一个密码**。
{% endhint %}

## 📖 简介

首次进服必须完成注册与登录，否则无法移动、聊天或使用菜单。账号与你的 **游戏 ID** 绑定，请妥善保管密码。

## 💡 安全小贴士

- 使用 **8 位以上** 混合密码，避免 `123456`  
- 不要与他人 **共享账号**，赞助也无法转移到其他 ID  
- 忘记密码请联系 QQ 群 **518445497**
