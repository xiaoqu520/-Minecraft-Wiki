# 资源包

安途使用 **ItemsAdder**（及 CraftEngine 等）提供自定义物品、菜单背景与支付界面材质。进服时客户端会提示 **下载服务器资源包**，建议选择 **同意/启用**。

> **[待补图 IMG-007]** 资源包下载提示弹窗

## 为什么需要资源包

- 主菜单（`/cd`）使用自定义图片 UI（ItemsAdder `img_menu_*`）  
- 点券充值界面（`paymenu`）使用自定义支付宝/微信样式  
- 部分物品、时装、称号展示依赖材质包  

未启用资源包时，可能看到 **紫黑方块**、错位界面或纯文字菜单，属正常现象，但体验会下降。

## 操作步骤

1. 连接服务器后，在提示框选择 **是** / **启用资源包**  
2. 等待下载完成（首次可能较慢）  
3. 若失败，断开重连；仍失败可尝试关闭 VPN 或换网络  
4. 可在单人世界先测试资源包是否被防火墙拦截  

## 跨版本玩家

使用 ViaVersion 进入时，资源包协议可能不完全兼容。建议主版本 **1.20.1**。

## 仍无法加载？

1. 检查客户端「服务器资源包」是否为启用  
2. 加入 QQ 群 **518445497** 并附上客户端版本与截图  

相关配置由管理在服务端维护，玩家无需手动安装 Mod（保持 **原版 + 资源包** 即可）。
