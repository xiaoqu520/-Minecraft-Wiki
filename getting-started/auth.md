# 注册与登录

安途使用 **AuthMe** 进行游戏内注册与登录。服务器为 **离线模式**（`online-mode=false`），即使用正版客户端也需设置 **独立游戏密码**，请勿与 Mojang 账号密码混用。

> **[待补图 IMG-005]** 首次进服 AuthMe 注册/登录提示界面

## 常用命令

| 命令 | 说明 |
|------|------|
| `/register <密码> <确认密码>` | 首次进服注册 |
| `/login <密码>` | 每次进服登录 |
| `/changepassword <旧密码> <新密码>` | 修改密码 |

具体子命令以游戏内 `/help` 或 AuthMe 提示为准。

## 注册流程

1. 首次进入服务器，等待 AuthMe 提示  
2. 输入 `/register 你的密码 你的密码`（两次密码需一致）  
3. 注册成功后自动或手动 `/login`  
4. 进服欢迎消息会引导你使用 `/plt open` 查看称号等  

## 安全须知

- **不要**将密码告诉他人或共用账号  
- **不要**使用过于简单的密码（如 `123456`）  
- 离线服账号与 **游戏 ID 绑定**，换 ID 等于新账号  
- 赞助、会员、点券均发放到当前游戏 ID，**充错号无法自动转移**  

## 与正版的关系

- 正版玩家仍需 AuthMe 密码  
- 使用相近版本连接时，请优先 **1.20.1**
