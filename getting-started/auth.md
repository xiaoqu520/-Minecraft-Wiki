# 🔐 注册与登录

{% hint style="info" %}
安途使用 **AuthMe** 独立游戏密码，与 Mojang 正版密码 **不是同一个**。
{% endhint %}

## 👤 适合谁

**首次进服** 或更换设备后需要登录的所有玩家。

## 📖 简介

未完成注册/登录时，你无法移动、聊天或打开菜单。账号与 **游戏 ID** 绑定；赞助、会员、点券均发到当前 ID，**充错无法转移**。

## ✨ 你能获得

- 安全的游戏内身份  
- 保护背包、建筑与赞助权益  

> **[待补图 IMG-005]** 首次进服 AuthMe 注册/登录提示界面

## 常用命令

| 命令 | 说明 |
|------|------|
| `/register <密码> <确认密码>` | 首次进服注册 |
| `/login <密码>` | 每次进服登录 |
| `/changepassword <旧密码> <新密码>` | 修改密码 |

## 🛤️ 注册流程

1. 首次进入服务器，等待 AuthMe 提示  
2. 输入 `/register 你的密码 你的密码`（两次一致）  
3. 注册成功后 `/login`  
4. 按提示继续 [资源包](资源包.md) 与 `/cd` 主菜单  

## ❓ 常见问题

| 问题 | 处理 |
|------|------|
| 忘记密码 | QQ 群 **518445497** 联系管理 |
| 能用正版密码吗 | ❌ 必须单独设置 AuthMe 密码 |
| 密码太简单 | 避免 `123456`，建议 8 位以上混合字符 |
| 和别人共用号 | ❌ 不建议，纠纷与赞助风险自负 |

## 与正版的关系

正版客户端也需 AuthMe；版本请优先 **1.20.1**。
