# 🪙 如何获得金币

{% hint style="success" %}
金币是服内最基础的 CMI 经济货币，**不花人民币** 也能通过玩法积累。
{% endhint %}

## 👤 适合谁

想免费发展、或想知道「怎么攒钱」的新人与日常玩家。

## ✨ 主要途径

| 途径 | 说明 | 文档 |
|------|------|------|
| 📅 每日签到 | 稳定小额收入 | [每日签到](../gameplay/每日签到.md) |
| 📋 每日任务 | 完成 6 条随机任务，金币量随稀有度上升 | [每日任务](../gameplay/每日任务.md) |
| 🛒 出售物品 | 系统商店 `shopgui` 或全球市场 QuickShop | [系统商店](../gameplay/shops.md) · [全球市场](../gameplay/market.md) |
| 🎁 活动/红包 | 节假日红包雨、周末游戏夜等 | [红包](../gameplay/红包.md) |
| 👑 会员加成 | VIP+ 出售物品有额外加成 | [会员总览](../membership/overview.md) |

{% hint style="info" %}
💡 会员 **提高折扣与出售加成**，不直接送大量金币；长期玩建议配合每日任务。
{% endhint %}

## 与点券的区别

金币 ≠ 点券。点券需 [人民币赞助](../donation/points.md)，见 [货币说明](currencies.md)。
