# 💸 消费与转账

{% hint style="info" %}
本章讲 **游戏金币**（CMI 经济）。花 **人民币** 买点券见 [赞助指南](../donation/hubs/赞助指南.md)。
{% endhint %}

## 👤 适合谁

已有金币、需要在商店购物或给好友转账的玩家。

## 💰 常见消费

| 场景 | 说明 |
|------|------|
| 🏪 系统商店 | `/shopgui`，会员享购买折扣 |
| 🛒 全球市场 | 购买其他玩家上架物品 |
| 🏷️ 称号 | 部分称号支持 **金币** 购买 |
| 🏡 领地 | Residence 按方块计费：`BuyCost` 约 **0.5～0.7** 金币/格（默认/VIP 组），续费 `RenewCost` 约 **0.2～0.3**；MVP 组可 **出售领地**（`CanSell: true`） |

## 🤝 玩家转账

CMI 经济支持玩家转账（`config.yml` 含 `/pay` 税收设置）：

```
/pay <玩家名> <金额>
```

具体是否收税、最低金额以游戏内 `/pay` 提示为准。

{% hint style="warning" %}
转账前确认对方 ID，谨防诈骗；私下交易纠纷服内不担保。
{% endhint %}
