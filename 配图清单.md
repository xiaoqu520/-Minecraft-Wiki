## GitBook 替换步骤

1. 在 GitBook 编辑器中打开对应页面  
2. 找到 `**[待补图 IMG-XXX]**` 引用块  
3. 插入图片并删除或保留 ID 注释（建议保留 ID 便于后续更换）  
4. 在本表「状态」列自行标注 ✅（本仓库 Markdown 不跟踪状态）
