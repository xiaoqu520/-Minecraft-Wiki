# 安途 Minecraft 服务器

欢迎来到 **安途** —— 以星海航路为主题的 Minecraft **1.20.1** 生存 RPG 服务器。在这里你可以探索多世界、建设领地、体验粘液科技与丰富社交玩法，并通过赞助支持公益服运营。

> **[待补图 IMG-001]** 安途主城或品牌横幅（建议 1920×400，用于 GitBook 封面）

## 快速加入

| 项目 | 内容 |
|------|------|
| **主地址** | `play.antu.fun` |
| **备用地址** | `play.antu.fun:3000`（主地址无法连接时尝试） |
| **版本** | Java 版 **1.20.1**（相近版本可通过 ViaVersion 尝试进入，不保证完整兼容） |
| **QQ 交流群** | **518445497** |

详细步骤见 [加入服务器](join.md)。

## 重要提示

- 本服务器为 **离线模式认证**（AuthMe），请设置强密码，勿与他人共享账号。  
- **会员为限时权益**，到期自动收回，非永久 Rank。  
- **点券充值**与 **会员月卡/季卡** 相互独立，详见 [点券充值](donation/points.md) 与 [会员总览](membership/overview.md)。
