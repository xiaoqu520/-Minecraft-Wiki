# 🚀 加入服务器

## 📖 简介

三步即可连接安途：添加服务器地址、注册登录、下载资源包。

## ✨ 你能获得

- 顺利进入主城并开始冒险。

{% hint style="info" %}
💡 **小贴士**：连不上请尝试备用地址 `play.antu.fun:3000`。
{% endhint %}

## 服务器地址

| 类型 | 地址 | 说明 |
|------|------|------|
| **主地址** | `play.antu.fun` | 推荐首选，使用默认端口 |
| **备用地址** | `play.antu.fun:3000` | 主地址超时、无法连接时再试 |

> **[待补图 IMG-002]** 客户端「添加服务器」界面，标注主地址与备用地址输入位置

## 客户端要求

- **游戏版本**：Minecraft Java **1.20.1**（与服务端 Paper 1.20.1 一致）
- **登录方式**：本服用 **AuthMe** 注册游戏内账号（非正版强制绑定），详见 [注册与登录](getting-started/auth.md)
- **跨版本**：服务端安装 ViaVersion，部分高/低版本可尝试进入，**不保证**资源包与玩法插件完全正常

## 添加服务器步骤

1. 启动 Minecraft Java 版，进入 **多人游戏** → **添加服务器**
2. **服务器名称**可填：`安途` 或任意名称
3. **服务器地址**填：`play.antu.fun`
4. 保存后尝试连接；若长时间无法连接，将地址改为 `play.antu.fun:3000` 再试
5. 首次进入后按提示 **注册**（`/register`）并 **登录**（`/login`）
6. 按提示 **下载并启用资源包**，见 [资源包](getting-started/资源包.md)
7. 输入 `/cd` 打开 [主菜单](getting-started/主菜单.md)

## 进服后第一件事

```
/register <密码> <确认密码>
/login <密码>
```

注册成功后会获得新手称号与启程礼包相关提示，可在主菜单继续探索。
