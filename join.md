# 🚀 加入服务器

## 👤 适合谁

准备在 Java 版 **1.20.1** 连接安途的所有玩家。

## 📖 简介

连接只需地址 + 注册 + 资源包；全程约 **5～15 分钟**（视下载速度）。

## 🖥️ 服务器地址

| 类型 | 地址 | 说明 |
|------|------|------|
| 🌐 **主地址** | `play.antu.fun` | ⭐ 首选 |
| 🔄 **备用** | `play.antu.fun:3000` | 主地址连不上再试 |

> **[待补图 IMG-002]** 添加服务器界面

## 🛤️ 首登时间线

| 步骤 | 做什么 |
|------|--------|
| 1️⃣ | 启动 **Java 1.20.1**，多人游戏 → 添加服务器 |
| 2️⃣ | 地址填 `play.antu.fun`，进入服务器 |
| 3️⃣ | `/register 密码 密码` → `/login 密码` |
| 4️⃣ | 同意下载 **资源包** |
| 5️⃣ | `/cd` 打开主菜单，开始探索 |
| 6️⃣ | [每日签到](../gameplay/每日签到.md) + [每日任务](../gameplay/每日任务.md) |
| 7️⃣ | 需要赞助时看 [赞助指南](../donation/hubs/赞助指南.md) |

## 🔧 连不上？故障排除

| 现象 | 建议 |
|------|------|
| 超时 / 无法连接 | 换备用地址 `:3000` |
| 版本不对 | 使用 **1.20.1** |
| 资源包失败 | 重连、检查防火墙，见 [资源包](getting-started/资源包.md) |
| 卡在登录界面 | 确认已 `/login`，见 [注册与登录](getting-started/auth.md) |

## 客户端要求

- **AuthMe** 游戏内账号（见 [注册与登录](getting-started/auth.md)）  
- **ViaVersion**：其他版本可尝试，不保证完整兼容
