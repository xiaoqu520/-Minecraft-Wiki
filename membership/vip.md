入门赞助档：更多家园、领地、商店折扣与好任务率。

**定位**：入门赞助档，提升生存品质与日常收益。

## 核心权益

- 称号 **「安途会员」**，TAB 显示 **\[VIP\]**
- **/home × 5**
- **领地 4 个**，单块约 **20×20**
- **VIP 每日礼包**（`/kit` 或菜单，见 [每日礼包](会员每日礼包.md)）
- **商店 8 折** 购买，出售 **+20%**
- **玩家店 10 个**
- 每日任务 **好任务率 64%**（见 [每日任务](../gameplay/每日任务.md)）

## 购买方式

主菜单 → 会员界面 → **VIP** → 选择月卡/季卡 → 微信或支付宝支付。

命令示例（支付流程由菜单触发）：

```
/cp buy vip_month wx
/cp buy vip_quarter zfb
```

（渠道 `wx` / `zfb` 以菜单为准）

## 到期说明

权限通过 `lp user <玩家> parent addtemp vip 30d`（或 90d）发放，**到期自动移除**。
