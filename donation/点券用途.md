# 🛍️ 点券用途一览

{% hint style="info" %}
点券通过 [赞助充值](points.md) 获得（1 元 = 10 点券），用于服内 **选择性消费**，不是万能货币。
{% endhint %}

## 👤 适合谁

已充值点券、想了解「钱花在哪」的玩家。

## ✨ 你能买到什么

| 类别 | 示例 | 文档 |
|------|------|------|
| 🏷️ 称号 | 点券类永久/限时称号 | [称号系统](../gameplay/titles.md) |
| 🐾 宠物 | 传说档点券专售（星航巨鲸等） | [宠物系统](../gameplay/pets.md) |
| 📍 地标 | 部分传送/地标功能 | [玩家地标](../gameplay/landmarks.md) |
| 🎁 宝箱 | 海风/星辰抽奖可能获得点券或消耗点券钥匙 | [宝箱抽奖](../gameplay/crates.md) |
| 🧧 红包 | 发放点券红包 | [红包系统](../gameplay/红包.md) |
| 🏪 混合商店 | 称号商城 `shop_pts_*` 等 | [称号系统](../gameplay/titles.md) |

{% hint style="warning" %}
点券 **不能** 替代 **会员月卡/季卡** 权限，也不能直接当「游戏金币」在 QuickShop 里通用（除非某玩法单独支持）。见 [会员与点券区别](会员与点券区别.md)。
{% endhint %}

## ❓ 常见问题

**点券会过期吗？**  
一般长期保留在账号上，具体以游戏内余额显示为准；活动赠送可能有期限。

**点券可以送人吗？**  
仅支持游戏内明确允许的玩法（如红包），请勿私下买卖账号或点券。
