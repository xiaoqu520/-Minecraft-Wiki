# 🔀 会员与点券区别

{% hint style="success" %}
一句话：**会员 = 限时权限包**；**点券 = 赞助货币**。各买各的，互不赠送。
{% endhint %}

## 👤 适合谁

准备花钱前，不确定该买「月卡」还是「充点券」的玩家。

## 📊 对照表

| 项目 | 👑 会员（VIP/SVIP/MVP） | 🎫 点券 |
|------|-------------------------|--------|
| **是什么** | LuckPerms **限时权限组** | PlayerPoints **虚拟货币** |
| **怎么获得** | 月卡/季卡人民币（CooPay 礼包） | 赞助充值 1 元=10 点券 |
| **时效** | 30/90 天，**到期收回** | 一般长期余额 |
| **典型用途** | /fly、领地数量、商店折扣、好任务率 | 称号、点券宠、红包、抽奖等 |
| **购买入口** | 主菜单 → 会员界面 | 主菜单 → 点券充值 / `/paymenu` |
| **能否互相替代** | ❌ 不能 | ❌ 不能 |

## 🛒 我该怎么选？

| 你的目标 | 建议 |
|----------|------|
| 想要飞行、更多 home、商店折扣 | 买 [会员](../membership/overview.md) |
| 想要某个点券称号或点券宠物 | 充 [点券](points.md) |
| 两个都想要 | **各买一次**，无打包优惠除非活动公告 |
