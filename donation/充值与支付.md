# 充值与支付步骤

安途使用 **CooPay** 处理微信、支付宝扫码赞助。请在 **游戏内** 完成，勿信第三方代充。

## 方式一：菜单（推荐）

1. 输入 `/cd` 打开主菜单  
2. **会员界面** → **点券充值**  
3. 或命令 `/paymenu` 打开充值选择  
4. 选择 **支付宝** 或 **微信**  
5. 选择预设金额（1 / 10 / 100 元等）或自定义 **整数** 金额  
6. 按提示扫码支付，等待点券到账

> **[待补图 IMG-043]** paymenu 支付宝充值页

## 方式二：命令

### 点券（任意整数金额，元）

```
/coopay trade wx <金额>
/coopay trade ZFB <金额>
```

别名：`/cp`、`/coo`（以游戏内为准）

示例：赞助 10 元 → 100 点券

```
/cp trade wx 10
```

### 会员礼包

```
/cp buy vip_month wx
/cp buy svip_quarter zfb
/cp buy mvp_month wx
```

礼包 ID：`vip_month`、`vip_quarter`、`svip_month`、`svip_quarter`、`mvp_month`、`mvp_quarter`

## 支付注意

| 项目 | 说明 |
|------|------|
| 金额 | 必须为 **整数**（元） |
| 超时 | 10 分钟内完成支付 |
| 到账 | 一般即时；未到账见 [客服](../support.md) |
| 移动 | 创建订单时 **不可移动**（防误触） |

## 会员购买路径

主菜单 → 会员界面 → 选择 VIP/SVIP/MVP → 月卡/季卡 → 选择支付方式 → 扫码。
