# 💳 充值与支付步骤

{% hint style="warning" %}
创建订单后 **请勿移动角色**；金额须为 **整数元**；10 分钟内完成支付。
{% endhint %}

## 👤 适合谁

已决定赞助点券、需要扫码支付教程的玩家。

## 📖 简介

安途使用 **CooPay** 在游戏内完成微信/支付宝赞助。**勿信第三方代充**。

---

## 🎮 方式一：菜单（推荐）

1. `/cd` → **会员界面** → **点券充值**（或 `/paymenu`）  
2. 选择支付方式（见下方 Tabs）  
3. 选择 **1 / 10 / 100 元** 或自定义整数金额  
4. 扫码支付，等待点券到账（一般即时）  

> **[待补图 IMG-042]** paymenu 微信/支付宝选择  
> **[待补图 IMG-043]** 支付宝充值页  
> **[待补图 IMG-044]** 微信充值页  

{% tabs %}
{% tab title="微信支付" %}
1. 在充值菜单选择 **微信**  
2. 输入 **整数** 金额（元）  
3. 扫描屏幕二维码  
4. 支付完成后等待点券到账  

命令快捷：`/cp trade wx <金额>` 或 `/coopay trade wx <金额>`
{% endtab %}
{% tab title="支付宝" %}
1. 在充值菜单选择 **支付宝**  
2. 输入 **整数** 金额（元）  
3. 扫描屏幕二维码  
4. 支付完成后等待点券到账  

命令快捷：`/cp trade ZFB <金额>`（别名 `/cp`、`/coo`）
{% endtab %}
{% endtabs %}

---

## ⌨️ 方式二：命令（点券）

示例：赞助 **10 元** → **100 点券**

```
/cp trade wx 10
/cp trade ZFB 10
```

## 👑 购买会员（非点券）

会员请走 **会员界面** 选择 VIP/SVIP/MVP 月卡/季卡，例如：

```
/cp buy vip_month wx
/cp buy mvp_quarter zfb
```

礼包 ID：`vip_month`、`vip_quarter`、`svip_month`、`svip_quarter`、`mvp_month`、`mvp_quarter`  
详见 [会员总览](../membership/overview.md)。
