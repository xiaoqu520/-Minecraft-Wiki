# 地皮系统

地皮（Plot）为 **PlotSquared** 提供的独立建造区域，与 **领地（Residence）** 不同：地皮适合规整分区建家，领地适合在生存世界自由圈地。

## 当前状态：筹备中

游戏内 NPC 与提示为：**「地皮世界暂未开放，敬请期待！」**

主菜单 → 世界旅行 → **进入地皮** 会打开地皮管理菜单框架，但 **世界入口尚未对全体玩家开放**。请勿轻信非官方消息声称已开放。

> **[待补图 IMG-010]** 地皮主菜单（Plot_Main_Page）或 NPC 暂未开放提示

## 开放后预期功能（菜单已配置）

文档根据现有 TrMenu 预设，开放后可能包含：

| 功能 | 说明 |
|------|------|
| 地皮列表 | 浏览、传送到地皮 |
| 地皮管理 | 旗帜、信任、拒绝列表 |
| 新玩家地皮 | 首次领取流程 |

具体规则以开放时群内公告为准。

## 现在想建家怎么办？

- 使用 [领地系统](../gameplay/residence.md) 在 **生存大陆** 圈地  
- 或使用 [家园系统](homes.md) 标记传送点
