# 🏡 家园系统

{% hint style="success" %}
**CMI / HuskHomes** + TrMenu **Player_Home**：世界旅行 → **传送家园**。
{% endhint %}

## 👤 适合谁

需要在生存世界设置多个固定传送点的玩家；会员档可拥有更多 `/home` 数量。

## 打开方式

主菜单 `/cd` → **世界旅行** → **传送家园**（TrMenu：`open: Player_Home`）

> **[待补图 IMG-009]** 家园列表 GUI

## 家园数量（会员菜单 lore）

| 档位 | `/home` 数量 |
|------|----------------|
| 普通 | 3 |
| VIP | 5 |
| SVIP | 8 |
| MVP | 12 |

限时会员到期后恢复为当前权限档。详见 [会员总览](../membership/overview.md)。

## 常用操作

- 设置 / 传送 / 编辑 / 删除以 **Player_Home**、**Player_Home_Edit** 菜单及 CMI 提示为准（常见命令如 `/sethome`、`/home`）

## 提示

- 勿设在岩浆、虚空边等危险处  
- 与 [地皮](plots.md)（暂未开放）为不同系统
