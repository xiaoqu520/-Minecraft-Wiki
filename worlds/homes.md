# 家园系统

家园（Home）让你在世界中设置固定传送点。安途通过 **CMI** 与 **HuskHomes** / **TrMenu 家园菜单** 提供管理界面。

**打开方式**：主菜单 → 世界旅行 → **传送家园**

> **[待补图 IMG-009]** 家园列表 GUI（Player_Home）

## 家园数量（按会员档）

| 档位 | `/home` 数量 |
|------|----------------|
| 普通玩家 | 3 |
| VIP | 5 |
| SVIP | 8 |
| MVP | 12 |

会员为 **限时**，到期后数量恢复为当前权限档。详见 [会员总览](../membership/overview.md)。

## 常用操作

- 在目标位置设置家园（具体命令以 CMI 为准，常见为 `/sethome` 或菜单内设置）  
- 通过菜单或 `/home <名称>` 传送  
- 编辑、删除家园在 **Player_Home_Edit** 子菜单中完成  

## 提示

- 请勿将家园设在危险区域（岩浆、虚空边）  
- 地皮世界与家园为不同系统，见 [地皮系统](plots.md)
