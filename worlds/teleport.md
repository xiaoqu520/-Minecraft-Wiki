# 🌍 世界旅行与传送

{% hint style="success" %}
主菜单 **世界旅行** 打开 **维度传送** 菜单：回城、家园、三世界 RTP、地皮入口。
{% endhint %}

## 👤 适合谁

需要在主城、生存、资源、活动世界之间切换的玩家。

## 打开方式

主菜单 `/cd` → **世界旅行**

> **[待补图 IMG-008]** 世界旅行菜单完整截图

## 菜单选项（TrMenu 核对）

| 选项 | 效果 |
|------|------|
| 回城 | `spawn` |
| 传送家园 | 打开 `Player_Home` |
| 进入生存大陆 | `betterrtp world world_survival` |
| 进入资源世界 | `betterrtp world world_resource` |
| 进入活动世界 | `betterrtp world world_activity` |
| 进入地皮 | 打开 `Plot_Main_Page`（见 [地皮](plots.md)） |

## 世界说明

| 世界 | 用途 |
|------|------|
| 主城 / 出生点 | 社交、NPC、商店 |
| **world_survival** | 长期生存建家 |
| **world_resource** | 采集；按公告周期重置 |
| **world_activity** | 活动、小游戏场地 |
| 地皮世界 | **暂未开放** |
| Galactifun 星际 | 粘液科技进阶（见 [粘液科技](../gameplay/slimefun.md)） |
