# 世界旅行与传送

**打开方式**：主菜单 `/cd` → **世界旅行**

> **[待补图 IMG-008]** 世界旅行菜单完整截图

## 菜单选项

| 选项 | 效果 |
|------|------|
| 回城 | `/spawn` — 回到主城 |
| 传送家园 | 打开家园 GUI（Player_Home） |
| 进入生存大陆 | 随机传送至 `world_survival`（BetterRTP） |
| 进入资源世界 | 随机传送至 `world_resource` |
| 进入活动世界 | 随机传送至 `world_activity` |
| 进入地皮 | 打开地皮菜单（见 [地皮系统](plots.md)） |

## 世界说明（概览）

| 世界类型 | 用途 |
|----------|------|
| **主城 / 出生点** | 社交、NPC、活动、商店集中区 |
| **生存大陆** | 长期建家、生存发展，**不刷新** |
| **资源世界** | 采集资源，按服内公告周期重置（见活动日程） |
| **活动世界** | 服务器活动、小游戏场地 |
| **星际 / Galactifun** | 粘液科技星际探索相关维度（进阶玩法） |
| **地皮世界** | 独立地皮（**暂未开放**） |

资源世界重置等事项以游戏内公告与 QQ 群 **518445497** 为准。

## 随机传送

野外随机传送使用 **BetterRTP**，部分会员可 **绕过冷却**，见 [随机传送](../gameplay/rtp.md)。
