# ⚔️ 公会系统

## 📖 简介

公会让你与伙伴共建势力，参与服内活动。

**PlayerGuild** 让你创建或加入公会，与朋友共建势力。

**打开方式**：主菜单 `/cd` → **公会**

```
/plg open
```

> **[待补图 IMG-018]** 公会主界面

## 功能概览

- 创建 / 加入 / 离开公会  
- 公会发展、成员协作（具体以插件 GUI 为准）  
- 服内日程含 **公会战预告** 等活动（见群内公告）  

## 提示

- 公会名请遵守 [服务器守则](../rules/服务器守则.md)  
- 涉及 Multiverse / WorldEdit 的公会建筑请尊重他人领地
