# ⚔️ 公会系统

{% hint style="success" %}
**PlayerGuild**：`/plg open` 或主菜单 **公会**；离开公会后 **3 小时** 冷却方可再加入。
{% endhint %}

## 👤 适合谁

想与固定队友共建势力、参与服内公会活动的玩家。

## 打开方式

| 方式 | 操作 |
|------|------|
| 主菜单 | `/cd` → **公会** |
| 命令 | `/plg open` |

> **[待补图 IMG-018]** 公会主界面

## 规则（`PlayerGuild/config.yml` 节选）

| 项目 | 配置 |
|------|------|
| 离开后再加入冷却 | **3 小时**（`joinCooling`） |
| 无公会登录提醒 | **开启** |
| 公会战结束回到 | 比赛前位置（`pvpEndModel: default`） |

## 服内活动

CMI 日程 **「公会战预告」** 广播会提示前往 `/plg open` 查看公会（具体时间以 `Schedules.yml` 与群内公告为准）。

## 提示

- 公会名遵守 [服务器守则](../rules/服务器守则.md)  
- 建筑请尊重他人 [领地](residence.md)
