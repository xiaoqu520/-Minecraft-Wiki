# 📍 玩家地标

{% hint style="success" %}
**PlayerWarp**：创建个人地标供他人付费传送；主菜单 **世界地标** 或 `/plw open`。
{% endhint %}

## 👤 适合谁

有店铺、景点、农场入口想分享给全服玩家的建设者。

## 打开方式

| 方式 | 操作 |
|------|------|
| 主菜单 | `/cd` → **世界地标** |
| 命令 | `/plw open` |

> **[待补图 IMG-021]** 地标列表界面

## 规则（`PlayerWarp/config.yml`）

| 项目 | 配置值 |
|------|--------|
| 默认可创建地标数 | **10** 个（`warpNumber.default`） |
| 传送税率 | **20%**（`taxRate: 0.2`） |
| 访问价格上限 | **10000**（`maxPrice`） |
| 重复传送计流量 | **1 小时**内同地标不重复计数（`tpNumberDate: 1`） |
| 禁止创建的世界 | `world_spawn`、资源世界等（见配置 `notWorld` 列表） |

权限节点可提升上限，例如 `playerWarp.number.vip1` → 20 个（配置中有 `vip1/vip2/vip3` 档位，是否与会员 VIP 同名对应以 LuckPerms 实配为准）。

## 传送延迟

| 档位 | 传送前等待（秒） | 冷却（秒） |
|------|------------------|------------|
| 默认 | 3 | 60 |
| vip1 权限 | 2 | 30 |
| vip2 权限 | 1 | 10 |
| vip3 权限 | 0 | 0 |

## 提示

- 创建前确认位置合法、不遮挡他人 [领地](residence.md)  
- 地标名称需遵守 [服务器守则](../rules/服务器守则.md)  
- 若支持点券相关消费，见 [货币说明](../economy/currencies.md)
