# 📍 玩家地标

## 📖 简介

创建地标供他人传送，可设于你的店铺或景点。

**PlayerWarp** 允许玩家创建地标供他人传送（可能消耗金币或点券，以服内配置为准）。

**打开方式**：主菜单 `/cd` → **世界地标**

```
/plw open
```

> **[待补图 IMG-021]** 地标列表界面

## 说明

- 创建地标前请选好安全、不侵权的位置  
- 若支持点券购买传送权限，见 [货币说明](../economy/currencies.md)  
- 滥用地标发布广告、违规内容将被处理，见 [服务器守则](../rules/服务器守则.md)
