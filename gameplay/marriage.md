# 💍 结婚系统

{% hint style="success" %}
**MythicMarry**：`/cd2` → 结婚菜单，或 `/marry`。
{% endhint %}

## 👤 适合谁

希望与另一位玩家建立游戏内伴侣关系、参与婚礼仪式的玩家。

## 打开方式

| 方式 | 操作 |
|------|------|
| 主菜单 | `/cd2` → **结婚菜单** |
| 命令 | `/marry` |

> **[待补图 IMG-020]** 结婚菜单界面

## 规则（`MythicMarry/config.yml` 节选）

| 项目 | 配置 |
|------|------|
| 每日求婚次数上限 | **3** 次 |
| 求婚最低在线时长 | **10** 小时 |
| 求婚费用 | **0**（免费） |
| 订婚有效期 | **48** 小时 |

婚礼流程、离婚、任务奖励等以游戏内 GUI 与 `tasks.yml` 为准。
