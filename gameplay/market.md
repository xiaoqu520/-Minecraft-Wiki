# 🛒 全球市场

{% hint style="success" %}
**QuickShop-Hikari**：玩家对玩家交易；玩法菜单或 `/market gui` 打开。
{% endhint %}

## 👤 适合谁

想买卖其他玩家物资、或自己摆摊赚金币的玩家。

## 打开方式

| 方式 | 操作 |
|------|------|
| 玩法菜单 | `/cd` → **玩法菜单** → **全球市场**（`market gui`） |
| 命令 | `/market gui` |

主菜单 **商城购物** 里的「全球市场」实际打开的是 **`/shopgui`**（系统商店），请勿混淆。

> **[待补图 IMG-022]** 全球市场购买界面

## 功能

| 功能 | 说明 |
|------|------|
| 购买 | 浏览玩家上架的箱子商店 |
| 个人邮箱 | 领取已购物品 |

## 与系统商店

| | 全球市场 | [系统商店](shops.md) |
|--|----------|----------------------|
| 插件 | QuickShop-Hikari | ShopGUIPlus |
| 折扣 | 以箱子标价为准 | 会员 `pricemodifiers` 折扣 |
