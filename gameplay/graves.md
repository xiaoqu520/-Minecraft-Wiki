# ⚰️ 死亡墓碑

## 📖 简介

死亡生成墓碑，保存掉落物便于取回。

**GravesX** 在玩家死亡时生成墓碑，保存掉落物，便于找回。

**打开方式**：主菜单 `/cd` → **死亡记录**

```
/grave gui
```

> **[待补图 IMG-027]** 墓碑列表 GUI

## 会员相关

MVP 档享有 `graves.permission.mvp` 等墓碑相关权益（保护、保留时间等以插件为准）。

## 提示

- 及时取回墓碑，避免过期或遭他人破坏（视领地/世界规则）  
- PVP 区域死亡风险自负
