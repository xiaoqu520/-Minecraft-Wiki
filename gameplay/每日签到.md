# 📅 每日签到

{% hint style="success" %}
**LiteSignIn**：`/cd2` → 签到，或 `/signin gui`；默认组每日奖励含钻石/金锭/铁锭（以 GUI 为准）。
{% endhint %}

## 👤 适合谁

每天上线约 **1 分钟**、想稳定领取基础物资的玩家。

## 打开方式

| 方式 | 操作 |
|------|------|
| 主菜单 | `/cd2` → **签到** |
| 命令 | `/signin gui` |

> **[待补图 IMG-012]** 签到 GUI 界面

## 默认奖励（`LiteSignIn/RewardSettings.yml` · Default 组）

| 物品 | 数量 |
|------|------|
| 钻石 | 1 |
| 金锭 | 3 |
| 铁锭 | 6 |

另有 **VIP** 权限组可配置更高奖励（`Groups-Priority` 含 VIP）。以游戏内当日显示为准。
