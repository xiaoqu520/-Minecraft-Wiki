# 🏪 系统商店

{% hint style="success" %}
**ShopGUIPlus**：`/shopgui` 或主菜单 → 商城购物；会员购买 **×0.80 / ×0.75 / ×0.70**，出售 **×1.20 / ×1.25 / ×1.30**（不含刷怪笼商店）。
{% endhint %}

## 👤 适合谁

想快速买卖常用物资、享受会员折扣的玩家。

## 打开方式

| 方式 | 操作 |
|------|------|
| 命令 | `/shopgui` |
| 主菜单 | `/cd` → **商城购物**（TrMenu 执行 `shopgui`） |

> **[待补图 IMG-023]** ShopGUIPlus 主商店界面

## 会员价格修正（`pricemodifiers.yml`）

对各分类商店（blocks、矿物、掉落物等）统一生效：

| 档位 | 购买倍率 | 出售倍率 |
|------|----------|----------|
| VIP | **0.80**（8 折） | **1.20**（+20%） |
| SVIP | **0.75**（7.5 折） | **1.25**（+25%） |
| MVP | **0.70**（7 折） | **1.30**（+30%） |

{% hint style="warning" %}
配置注释写明：**不含 spawner（刷怪笼）商店**。该分类以游戏内标价为准。
{% endhint %}
