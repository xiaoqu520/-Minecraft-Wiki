# 🏪 系统商店

## 📖 简介

ShopGUIPlus 官方商店，会员享折扣与出售加成。

**ShopGUIPlus** 提供服务器官方商店，出售/收购各类物品。

**打开方式**：主菜单 → 商城购物 → **全球市场** 旁路为 `shopgui`；或：

```
/shopgui
```

> **[待补图 IMG-023]** ShopGUIPlus 主商店界面

| 档位 | 购买折扣 | 出售加成 |
