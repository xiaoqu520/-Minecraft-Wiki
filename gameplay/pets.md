# 🐾 宠物系统

## 📖 简介

**CompanionsPlus** 提供战斗辅助宠物，陪伴探索与战斗。部分 **传说档** 支持点券专售（安途舰长、星航巨鲸等），是长期玩家的收藏向内容。

## ✨ 你能获得

- 战斗被动加成与随行特效  
- 多主题宠物线：海港、主世界、下界、点券传说  

{% hint style="info" %}
💡 **小贴士**：先 [充值点券](../../donation/points.md) 再购买点券档宠物；仓库与商店命令见上。
{% endhint %}

## 🎮 常用命令

| 命令 | 说明 |
|------|------|
| `/cpp shop` | 宠物商店（主菜单 → 商城购物 → 宠物召唤） |
| `/companionsplus open` | 宠物仓库（`/cd2`） |

## 🌟 档位概览

| 系列 | 说明 |
|------|------|
| tier_c_* | 入门伙伴（海港、主世界、下界主题） |
| tier_a / tier_b | 进阶宠物 |
| tier_s | **点券专售** 传说（星航巨鲸、深渊海妖等） |
| tier_ss_points / tier_sss_points | 安途舰长、安途创世等收藏向 |

具体价格与技能以游戏内商店为准。

## ⚠️ 消费提示
