# 🐾 宠物系统

{% hint style="success" %}
**CompanionsPlus**：`/cpp shop` 购买 · `/companionsplus open` 仓库 · 主菜单商城 → 宠物召唤。
{% endhint %}

## 👤 适合谁

想要战斗随从、愿意用金币或 [点券](../donation/points.md) 购买收藏向宠物的玩家。

## 你能获得

- 战斗辅助与随行效果（以各宠物配置为准）  
- 多档位宠物线（见下表概览）

## 🎮 常用命令

| 命令 | 说明 |
|------|------|
| `/cpp shop` | 宠物商店（主菜单 → 商城购物 → 宠物召唤） |
| `/companionsplus open` | 宠物仓库（`/cd2`） |

> **[待补图 IMG-016]** 宠物商店界面  
> **[待补图 IMG-017]** 宠物仓库界面

## 🌟 档位概览

| 系列 | 说明 |
|------|------|
| tier_c_* | 入门伙伴（海港、主世界、下界主题） |
| tier_a / tier_b | 进阶宠物 |
| tier_s | **点券专售** 传说（星航巨鲸、深渊海妖等） |
| tier_ss_points / tier_sss_points | 安途舰长、安途创世等收藏向 |

具体价格与技能以游戏内商店为准。

## ⚠️ 消费提示

- 点券宠物需先 [充值点券](../donation/points.md)  
- 虚拟道具一般不支持退款，见 [赞助条例](../legal/赞助条例.md)
