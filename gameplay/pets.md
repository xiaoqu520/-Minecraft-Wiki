# 宠物系统

**CompanionsPlus** 提供战斗辅助宠物，部分档位支持 **点券专售**（安途舰长、星航巨鲸·点券等）。

| 命令 | 说明 |
|------|------|
| `/cpp shop` | 宠物商店（主菜单 → 商城购物 → 宠物召唤） |
| `/companionsplus open` | 宠物仓库（`/cd2`） |

> **[待补图 IMG-016]** 宠物商店界面  
> **[待补图 IMG-017]** 宠物仓库界面

## 档位概览

| 系列 | 说明 |
|------|------|
| tier_c_* | 入门伙伴（如海港、主世界、下界主题） |
| tier_a / tier_b | 进阶宠物 |
| tier_s | 含 **点券专售** 传说档（星航巨鲸、深渊海妖等） |
| tier_ss_points / tier_sss_points | 安途舰长、安途创世等高点券收藏向 |

具体价格与技能以游戏内商店为准。

- 点券宠物需先 [充值点券](../donation/points.md)
