# 🎰 赌场

{% hint style="warning" %}
**理性娱乐**：Vegas 为随机输赢小游戏；未成年人请在监护人同意下参与，见 [赞助条例](../legal/赞助条例.md)。
{% endhint %}

## 👤 适合谁

已开通 **SVIP 及以上** 限时会员、希望在主城娱乐区体验 Vegas 的玩家。

## 开放条件（可核对）

TrMenu **会员界面** 中 **SVIP** 档位说明写明：

- **「赌场全开」**

文档**不**描述 VIP / 普通玩家的赌场权限（配置中无对应说明）。若进不去或提示无权限，以游戏内提示为准。

## 插件说明

**Vegas** 提供赌场类玩法（老虎机、轮盘等具体机台以主城实装为准）。使用 **游戏金币** 结算（插件消息中的 `$` 为经济余额）。
