# 🎣 钓鱼玩法

{% hint style="success" %}
**PyroFishingPro** 增强钓鱼；每月首周 CMI 广播 **海风钓鱼季**，提示 `/fish menu`。
{% endhint %}

## 👤 适合谁

喜欢收集鱼种、参与限时钓鱼活动的玩家。

## 打开方式

以游戏内引导为准；活动广播示例：

```
/fish menu
```

（见 CMI `Schedules.yml` → `安途_海风钓鱼季_每月首周`）

> **[待补图 IMG-025]** 钓鱼界面或特殊鱼获

## 海风钓鱼季

| 项目 | 说明 |
|------|------|
| 触发 | CMI 日程 **每月首周**（具体日以 `Schedules.yml` 为准） |
| 广播文案 | 提示本周钓鱼挑战、`/fish menu` 查看图鉴与奖励 |
