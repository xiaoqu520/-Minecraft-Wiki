# 称号系统

**PlayerTitle** 提供称号展示、仓库与商城。称号可显示在 TAB、聊天等位置（占位符 `%playerTitle_use%`）。

| 命令 | 说明 |
|------|------|
| `/plt open` | 称号仓库，选择佩戴 |
| `/plt shop` | 称号商城 |
| `/plt menu` | 功能菜单（若已配置） |

> **[待补图 IMG-015]** 称号商城 `/plt shop`

## 会员绑定称号

随 LuckPerms 权限自动获得（到期收回）：

| 档位 | 称号 | TAB 标识 |
|------|------|----------|
| 普通 | 安途旅人 | — |
| VIP | 安途会员 | [VIP] |
| SVIP | 星航贵族 | [SVIP] |
| MVP | 至尊舰长 | [MVP] |

## 商城分类（概览）

| 类型 | 说明 |
|------|------|
| shop_gold_* | 金币购买 |
| shop_pts_* | 点券购买 |
| shop_dual_* | 双货币 |
| redeem_coin_* | 称号币兑换 |
| redeem_item_* | 物品兑换 |
| limited_* | 限定称号（活动发放） |

## 货币

- **称号币**：部分称号兑换，管理可用 `/plt coin give` 发放  
- **点券**：见 [点券说明](../donation/points.md)  

## 提示

- 进服欢迎会引导新玩家使用 `/plt open`  
- 购买付费称号前请确认 [赞助条例](../legal/赞助条例.md) 与理性消费
