# 👫 好友系统

{% hint style="success" %}
**HyronicFriends**：主菜单 → 好友系统，或命令 `hyronicfriends`。
{% endhint %}

## 👤 适合谁

希望长期一起玩、方便社交与组队的玩家。

## 打开方式

| 方式 | 操作 |
|------|------|
| 主菜单 | `/cd` → **好友系统** |
| 命令 | `hyronicfriends`（TrMenu 绑定） |

> **[待补图 IMG-019]** 好友列表界面

## 功能

- 添加 / 删除好友  
- 查看好友列表  
- 数据存于服务端数据库（配置为 MySQL），同一游戏 ID 换设备仍保留  

具体子命令以游戏内帮助为准。
