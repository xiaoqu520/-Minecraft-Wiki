# 👫 好友系统

## 📖 简介

添加好友，方便组队与社交。

**HyronicFriends** 提供好友列表、添加与删除。

**打开方式**：主菜单 `/cd` → **好友系统**

```
/hyronicfriends
```

> **[待补图 IMG-019]** 好友列表界面

## 功能

- 添加好友  
- 查看好友列表  
- 删除好友  

数据存储在服务端，换设备登录同一游戏 ID 仍保留。
