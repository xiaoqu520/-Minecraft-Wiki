# 🧧 红包系统

{% hint style="info" %}
**WarZHongBao**：发红包用 `/fhb`；聊天栏抢口令红包。过期 **600 秒**未领完退回。
{% endhint %}

## 👤 适合谁

想发全服福利、节日互动或公会活动的玩家（需对应权限）。

## 常用命令（`lang_zh.yml`）

帮助提示为 `/hb`；**发放**命令如下：

| 类型 | 命令格式 | 权限节点 |
|------|----------|----------|
| 金币红包 | `/fhb money <金额> <数量> [口令] [玩家]` | `bm.money` |
| 点券红包 | `/fhb ticket <金额> <数量> [口令] [玩家]` | `bm.ticket` |
| 物品红包 | `/fhb item [口令] [玩家]` | `bm.item` |

不填玩家名为 **公开红包**；口令红包需在聊天栏输入口令领取。

## 规则（`config.yml`）

| 项目 | 值 |
|------|-----|
| 单次最多红包份数 | **50**（`-1` 为不限制，当前为 50） |
| 最小金币 / 点券 | **1** |
| 过期退回 | **600 秒** |
| 发包冷却 | **0 秒**（无冷却） |
| 税收 | **关闭**（`tax.enabled: false`） |

## 活动

CMI 日程可配置 **定时红包**（`Schedules.yml` 示例多为注释状态）；节假日活动以 QQ 群 **518445497** 与游戏内广播为准。
