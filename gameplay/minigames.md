# 🕹️ 大堂小游戏

{% hint style="info" %}
主城娱乐区 **PocketGames / LobbyGames**；周末日程 **大堂游戏夜** 随机发 **500 金币** 参与奖。
{% endhint %}

## 👤 适合谁

想轻松社交、不参与高强度 PVP 的玩家。

## 在哪里玩

前往主城 **娱乐区**（具体坐标以城内路标为准），使用现场装置或 NPC 进入小游戏。

> **[待补图 IMG-031]** 大堂小游戏区域

## 周末活动（CMI `Schedules.yml`）

**安途_周末大堂游戏夜** 配置示例：

- 广播：周末大堂游戏夜开启，前往主城娱乐区  
- 奖励：`cmi money give [randomPlayer] 500`（随机一名在线玩家 **500 金币**）

实际触发星期与时间以 `Schedules.yml` 当前配置为准。
