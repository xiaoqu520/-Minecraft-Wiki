# ⚙️ 粘液科技

## 📖 简介

粘液科技 + 星际扩展，从发电机到跨星球探索。

## ✨ 你能获得

- 自动化生产与终局挑战。

安途以 **Slimefun** 为核心，搭配大量中文扩展，形成科技生存与星际探索体验。

> **[待补图 IMG-024]** 粘液科技研究台或主城科技区实景

## 主要插件生态

| 模块 | 说明 |
|------|------|
| **Slimefun** | 基础科技线、发电机、机器 |
| **Galactifun** | 星际行星（火星、月球、土卫六等维度） |
| **InfinityExpansion / LiteXpansion** | 终局扩展内容 |
| **Bump / Supreme / SlimeTinker** | 装备、强化等玩法 |
| **Gastronomicon** | 烹饪扩展 |
| **PyroFishingPro** | 高级钓鱼（见 [钓鱼玩法](fishing.md)） |
| **BetterFarming** | 种植增强 |

聊天前缀示例：`安途粘液科技`。

## 入门建议

1. 在主城或生存区建立基地  
2. 按 Slimefun 指南书制作研究台与基础机器  
3. 逐步解锁电力、自动化  
4. 进阶后探索 **Galactifun** 星际世界（需相应进度与装备）
