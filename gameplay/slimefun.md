# ⚙️ 粘液科技

{% hint style="success" %}
**Slimefun** 为核心科技线；服内日志显示约 **1260** 条配方（以当前版本为准）。进阶含 **Galactifun** 星际维度。
{% endhint %}

## 👤 适合谁

愿意长期投入、建设自动化工厂与星际探索的进阶玩家。

## 你能获得

- 发电机、加工机、自动化产线  
- **Galactifun** 等多世界探索（需逐步解锁进度）

> **[待补图 IMG-024]** 粘液科技研究台或主城科技区实景

## 主要插件生态

| 模块 | 说明 |
|------|------|
| **Slimefun** | 基础科技线、发电机、机器 |
| **Galactifun** | 星际行星（火星、月球、土卫六等维度） |
| **InfinityExpansion / LiteXpansion** | 终局扩展内容 |
| **Bump / Supreme / SlimeTinker** | 装备、强化等玩法 |
| **Gastronomicon** | 烹饪扩展 |
| **PyroFishingPro** | 高级钓鱼（见 [钓鱼玩法](fishing.md)） |
| **BetterFarming** | 种植增强 |

聊天前缀示例：`安途粘液科技`。

## 入门建议

1. 在主城或生存区建立基地  
2. 按 Slimefun 指南书制作研究台与基础机器  
3. 逐步解锁电力、自动化  
4. 进阶后探索 **Galactifun** 星际世界（需相应进度与装备）
