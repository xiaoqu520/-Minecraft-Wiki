# 🛡️ 领地系统

{% hint style="success" %}
**Residence** 圈地保护建筑与容器；主菜单 **领地系统** 打开 TrMenu 管理界面。
{% endhint %}

## 👤 适合谁

在生存世界建家、需要防拆防偷的玩家。

## 打开方式

主菜单 `/cd` → **领地系统** → 领地列表等子菜单。

> **[待补图 IMG-011]** 领地列表菜单

## TrMenu 子功能

| 菜单 | 用途 |
|------|------|
| 领地列表 | 查看、传送到自己的领地 |
| 领地成员管理 | 添加/移除成员 |
| 领地黑名单管理 | 禁止特定玩家进入 |
| 领地设置 | 规则、传送点等 |
| 领地权限 | 建造、容器等细粒度开关 |

## 会员领地限额（`Residence/groups.yml` + 会员菜单 lore）

| 档位 | 领地数量 | 平面尺寸 | 出售领地 |
|------|----------|----------|----------|
| 默认玩家 | 3 | 16×16 | 否 |
| VIP（`NextGroup`） | 4 | 20×20 | 否 |
| SVIP | 5 | 24×24 | 否 |
| MVP | 6 | 32×32 | **是**（`CanSell: true`） |

## 创建费用（金币/格）

| 组 | BuyCost | RenewCost |
|----|---------|-----------|
| Default | 0.5 | 0.2 |
| VIP（NextGroup） | 0.7 | 0.3 |
| SVIP | 0.6 | 0.25 |
| MVP | 0.55 | 0.22 |

以游戏内创建领地时的扣费提示为准。

## 与地皮区别

- **领地**：生存世界自选范围圈地  
- **地皮**：PlotSquared 固定网格 → [地皮系统](../worlds/plots.md)（**暂未开放**）
