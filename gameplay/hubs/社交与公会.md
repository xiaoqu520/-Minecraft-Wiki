# 👥 社交与公会

{% hint style="info" %}
与好友、伴侣、公会成员一起玩，可共享 [玩家地标](../landmarks.md) 与领地协作。
{% endhint %}

## 👤 适合谁

希望与朋友长期一起玩、建立势力、分享传送点的玩家。

## 📖 简介

好友、结婚、公会、地标四大社交系统。

⏱️ **预计耗时**：开户约 **5～15 分钟**；日常维护视公会活动而定

🔗 **会员关联**：地标数量可通过 `playerWarp.number.*` 权限提升（见 [玩家地标](../landmarks.md)）；与 VIP 会员组是否一一对应以 LuckPerms 为准。
