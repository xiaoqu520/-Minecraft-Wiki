## 常见问题该找谁

| 问题类型 | 建议操作 |
|----------|----------|
| 无法连接服务器 | 先尝试备用地址 `play.antu.fun:3000`，见 [加入服务器](join.md) |
| 忘记 AuthMe 密码 | 群内联系管理，按服内找回流程处理 |
| 赞助未到账 / 重复扣款 | 72 小时内带 **支付凭证、游戏 ID、大致时间** 联系管理，见 [赞助条例](legal/赞助条例.md) |
| 举报作弊、辱骂、破坏 | 提供证据（截图/录像）、对方 ID、时间，群内或游戏内联系在线管理 |
| 会员到期、权益疑问 | 会员为 **限时自动收回**，见 [会员总览](membership/overview.md) |

## 赞助与申诉流程

1. 确认使用 **游戏内官方菜单** 支付（微信 / 支付宝），勿信第三方代充  
2. 支付成功后一般 **即时到账**；点券或会员未到账请保留订单记录  
3. 加入 QQ 群 **518445497**，私信或 @管理 说明情况  
4. 处理原则以 [赞助与消费条例](legal/赞助条例.md) 为准  

**客服 QQ 群：518445497**（会员与点券页面亦会标注此群号）

## 网站联动说明

游戏内 **AntuWebBridge** 预留了与外部网站 API 的联动；当前 **网站功能筹备中**，请以游戏内菜单与本文档为准。上线后会在群内公告。

## 非支持范围

- 私下交易纠纷（服内鼓励使用官方商店与市场）  
- 因共享账号、密码泄露导致的损失  
- 违反 [服务器守则](rules/服务器守则.md) 导致的封禁申诉（需说明情况，管理保留最终决定权）
